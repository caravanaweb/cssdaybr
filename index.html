A importância de um frontend para sua aplicação by uma desenvolvedora backend

O desenvolvimento do front-end para um desenvolvedor back-end é uma coisa chata de se realizar, estamos sempre acostumados a focar na implementação e funcionalidade que quando vamos partir para o front-end acabamos definindo que é uma atividade bastante chata de ser realizada e ás vezes, deixando sempre pra lá. Só que esquecemos que além da funcionalidade ser muito importante como o aplicativo vai ser visto pelos clientes e usuários é tão importante quanto ele ser funcional. No talk, vou comentar os desafios de seguir uma guideline própria na construção de uma aplicação, mostrando o case Mete a Colher, e como podemos tornar o processo de desenvolvimento do frontend mais divertido para um desenvolvedor backend.

Aplicações Customizáveis com CSS Variables e Django

Customização é uma feature bem interessante que pode ser adicionada em diferentes tipos de cenários: desde aplicações white label até produtos SaaS. Nesta palestra, apresentarei conceitos envolvidos na stack de desenvolvimento de uma aplicação customizável e mostrando como implementar essa feature em um app Django usando CSS Variables.

A fantástica fábrica de CSS: Como começar a usar SASS no seu próximo projeto.

SASS é um pré processador de CSS e com ele você vai poder construir um código mais limpo e otimizado, criar estruturas reutilizáveis e definir variáveis para cores e medidas, por exemplo. É uma ferramenta poderosa que promove um aumento de produtividade. Você vai descobrir como iniciar seu próximo projeto usando essa nova forma de escrever CSS.

O que desenvolvedores deveriam aprender sobre design?

Apesar de designers e devs viverem em overlapping worlds, suas atividades são muito distintas. Compartilhar habilidades e aprender a falar a língua do outro é essencial para comunicar ideias. Por sorte, boa parte do trabalho de um desenvolvedor já é um trabalho de design. Não importa a tecnologia usada, boas decisões de design constroem softwares melhores e mais usáveis. Esta palestra tem como objetivo aumentar a sua consciência de design. Isto significa entender inicialmente o que um “bom design” significa. Em seguida iremos falar sobre como o processo de design pode ser incorporado no de desenvolvimento de software, além de discutir conceitos como consistência, prevenção de erros, reconhecimento e outros.
